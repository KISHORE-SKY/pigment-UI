Pigment UI:
  Pigment UI is a powerful CSS style generator that helps developers quickly create and customize UI styles. It provides ready-to-     use code in both CSS and Tailwind CSS, allowing you to copy and integrate styles seamlessly into any project.
  The goal of Pigment UI is to streamline the UI development workflow by making style creation faster, easier, and more consistent.


Features(Style Generators):
1.Color Picker
  Generate colors in multiple formats: HEX, RGB, RGBA, HSL
2.Gradient Generator
  Create both linear and radial gradients
3.Box Shadow Generator
  Customize shadows with full control over offsets, blur, spread, and color
4.Custom Button Generator
  Design reusable button styles with instant code output
5.Glassmorphism Generator
  Easily create modern glass-like UI effects
6.Predefined Forms
  Ready-made form UI components for faster development

Tech Stack:
  React JS
  CSS3
  Tailwind CSS
  React Router
  Web API(Clipboard API)
